## Quick Shopify Product Creation Checklist

For each product:

**Step 1: Basic Setup**
- [ ] Enter title, handle, product type
- [ ] Write description
- [ ] Set vendor
- [ ] Add to collections (2: main + category)
- [ ] Add tags

**Step 2: Variants**
- [ ] Create 3 options: Size, Style, Orientation
- [ ] Set prices: A4=£35, A3=£55, A2=£75
- [ ] Generate all 18 combinations
- [ ] Add SKUs (optional)

**Step 3: Images**
- [ ] Upload 6 images
- [ ] Set proper alt text for each
- [ ] Arrange in logical order

**Step 4: Metafields**
- [ ] Fill in all 9 metafields
- [ ] Verify data accuracy
- [ ] Check spelling and formatting

**Step 5: Template**
- [ ] Assign product.sportive-print template
- [ ] Preview product page
- [ ] Test variant switching
- [ ] Verify metafields display

**Step 6: Publish**
- [ ] Set product as active
- [ ] Verify in collection
- [ ] Test add to cart
- [ ] Check mobile display
