## Development Setup

### Prerequisites
- Node.js (v14+)
- npm or yarn
- Shopify Partner account
- Access to development store

### Installation
1. Install Shopify CLI: `npm install -g @shopify/cli @shopify/theme`
2. Clone this repository: `git clone [repository-url]`
3. Navigate to theme directory: `cd [theme-directory]`
4. Start development server: `shopify theme dev --store=[your-store]`

### Branch Strategy
- `main`: Production-ready code
- `staging`: Pre-production testing
- `development`: Active development

### Deployment
```bash
# Deploy to development theme
shopify theme push --store=[store] --theme=[theme-id]
```
```
